# Echo AI Companion 🌿

Echo AI is a chill, compassionate, and supportive AI companion designed to help users feel heard and uplifted. 

## Features
- **Multi-Threaded Conversations**: Manage multiple independent chat sessions simultaneously.
- **Real-Time Streaming**: Responsive message delivery with a natural "typing" effect.
- **Compassionate Persona**: Built-in personality focused on empathy and support.
- **Modern UI**: Clean, mobile-responsive violet theme built with Tailwind CSS.

## Tech Stack
- **Frontend**: React, Vite, Tailwind CSS, Lucide Icons
- **Backend**: Python, NextToken SDK (Gemini 2.0 Flash)
- **State Management**: React Hooks & File-based persistent storage

## Getting Started

### Prerequisites
- Node.js (v18+)
- Python 3.10+
- NextToken API Key

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/Lanor-Jephthah1/Echo-AI-Companion.git
   ```
2. Setup Frontend:
   ```bash
   cd frontend
   npm install
   npm run build
   ```
3. Setup Backend:
   ```bash
   pip install nexttoken
   python backend/main.py
   ```
